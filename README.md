# 简讯

<br>

## 这是什么？
这是一个为寄宿制普通高中学校提供的新闻浏览工具，方便学校在管控一体机不被用于娱乐的同时提供新闻信息。

<br>

## 注意
本项目部分功能使用公共 API 完成。

<br>

## 访问与地区限制
如果您所在的学校位于中国福建省、海南省、台湾省，可能因为网络环境而无法访问此项目提供的 URL 。

<br>
